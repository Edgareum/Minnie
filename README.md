# 感谢与祝福 · 致亲爱的同事们

A beautiful farewell & retirement message website for Chinese colleagues, built with pure HTML/CSS/JS.

## 🚀 Deploy to GitHub Pages

### Step 1: Push to GitHub

```bash
git add .
git commit -m "Add farewell website"
git push origin main
```

### Step 2: Enable GitHub Pages

1. Go to your repository on GitHub
2. Click **Settings** → **Pages**
3. Under **Source**, select **Deploy from a branch**
4. Choose branch: `main`, folder: `/public`
5. Click **Save**

### Step 3: Get your URL

Your site will be live at:
```
https://<your-username>.github.io/<repo-name>/
```

### Step 4: Generate QR Code

Once deployed, open the site and scroll to the bottom. Paste your GitHub Pages URL into the QR input field and click **生成 QR** to get a scannable QR code.
